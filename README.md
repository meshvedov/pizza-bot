Напиши пошаговый план по созданию web приложения на основе StreamLit.
Этот проект представляет собой умную RAG-систему, придуманную для ускорения сбора заказа. 
Бот под капотом будет иметь LLM модель,  соберет корзину сам, используя технику Few-Shot.

Фрейморки: Langchain + StreamLit.
База данных для RAG будет загружаться с сайта: https://dodopizza.ru/moscow
Примеры для Few-shot: KnowledgeBase4FewShot.csv


Пример 1 работы модели:
Запрос в чате:
Можно мне сок "Добрый" и две маленьких пиццы "Пепперони"?

Ответ модели: 
Ваш заказ:
- сок "Добрый"  - 1 х 150 рублей
- пицца "Пепперони" (25 см)  2 х 350 рублей
Итого: 850 рублей

Пример 2:
Запрос в чате:
Хочу что-нибудь сладкое и кофе

Ответ модели: 
Ваш заказ:
- Кофе "Латте" - 1 х 160 рублей
- Чизкейк - 1 х 210 рублей
Итого: 370 рублей
